# XSOC 智能运营平台 项目元信息
## 🎯 项目核心定位
> 仅实现blueprint中的【智能运营层】，不涉及上游安全组件、XDR/SOC层开发
- **输入触发点**：两种输入源：
  1. XDR中心主动推送的告警事件
  2. 用户手动输入的告警信息
- **处理流程**：严格遵循blueprint流程：
  `事件研判专家 → 溯源分析专家 → 风险处置专家 → 数据可视化专家`
## ✨ 核心特性
1. **四智能体协作**：四个专业安全专家智能体分工协作
2. **风险评分+ HITL机制**：风险度0-10分，高于阈值（默认7分，可配置）触发人工确认
3. **双模式能力接入**：
   - 本地API封装为Skills调用（XDR/资产/NDR/EDR/钉钉等）
   - 云端微步威胁情报通过MCP协议接入
4. **全链路留痕**：所有操作、决策、人工确认全程记录
5. **双维度归档**：XDR中心回写归档 + 钉钉AI表格可视化
