# XSOC 智能安全运营平台
## 🎯 项目定位
本项目仅实现**智能运营层**，完全基于blueprint设计，不涉及上游安全检测组件、XDR/SOC层的开发。
### 触发输入
仅支持两种输入源：
1. XDR安全运营中心主动推送的告警事件
2. 用户手动输入的告警信息
### 处理流程
严格遵循四步专家协作流程：
```
事件研判专家 → 溯源分析专家 → 风险处置专家 → 数据可视化专家
```
## ✨ 核心特性
- 🔹 **多智能体协作**：四位专业安全专家智能体分工协作，各司其职
- 🔹 **风险动态评估**：处置前自动进行0-10分风险度评分，高于阈值触发HITL人工确认
- 🔹 **双模式能力接入**：本地API封装为Skills调用 + 云端能力通过MCP协议接入
- 🔹 **全链路可追溯**：所有操作、决策、人工确认全程留痕，便于审计复盘
- 🔹 **双维度归档**：XDR中心结构化存储 + 钉钉AI表格可视化展示
